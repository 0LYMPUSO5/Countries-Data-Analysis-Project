🌍 Countries Data Analysis Project

📌 Overview
This project focuses on analyzing a dataset of countries to extract
meaningful insights related to population, democracy scores, and
regional distribution. It demonstrates basic data analysis skills using
Python and Pandas.

🎯 Objectives
-   Identify countries with the highest and lowest population
-   Analyze democracy scores across countries
-   Filter and explore data based on regions
-   Perform basic data cleaning and exploration

🛠️ Tools & Technologies
-   Python 🐍
-   Pandas 📊
-   Jupyter Notebook 📓

📂 Dataset
The dataset used in this project contains information about: - Country
names - Population - Democracy scores - Regions
Make sure ‘Countries.csv’ is in the same directory as the notebook.

🔍 Key Analysis Performed
-   Data loading and inspection using .info() and .describe()
-   Sorting countries based on population and democracy score
-   Filtering countries by region
-   Finding specific patterns in country names

📊 Sample Insights
-   Identified top countries by population
-   Found countries with highest and lowest democracy scores
-   Explored regional differences in data

🚀 How to Run
1.  Clone this repository.
2.  Navigate to the project folder: cd countries-data-analysis
3.  Open the Jupyter Notebook: jupyter notebook Countries.ipynb
